## Artikel 4

*Bij verordening kan in afwijking van artikel 12, tweede lid, onderdeel d, van de Uitvoeringsregeling het totale bedrag*

[wetbron](https://wetten.overheid.nl/BWBR0047157/2022-09-17#Artikel4)

### Definities (constanten uit wettekst)

- `verhoging_cap_echtgenoten` = **200000** — Maximale verhoging (eurocent) van de vrijstelling financiele middelen voor de belastingschuldige en echtgenoot, artikel 4, onder a.
- `verhoging_ratio_alleenstaande` = **0.75** — Ratio van de verhoging voor een alleenstaande ten opzichte van echtgenoten (artikel 4, onder b). 75%.
- `verhoging_ratio_alleenstaande_ouder` = **0.9** — Ratio van de verhoging voor een alleenstaande ouder ten opzichte van echtgenoten (artikel 4, onder c). 90%.

### Formules

```
verhoging_max  ≡
    IF huishoudtype = "echtgenoten"
             THEN verhoging_cap_echtgenoten
             ELSE IF huishoudtype = "alleenstaande_ouder"
             THEN verhoging_cap_echtgenoten × verhoging_ratio_alleenstaande_ouder
             ELSE IF huishoudtype = "alleenstaande"
             THEN verhoging_cap_echtgenoten × verhoging_ratio_alleenstaande
             ELSE 0
```
